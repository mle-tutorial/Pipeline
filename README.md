# Database

## 환경

- `pip install -r requirements.txt` 로 prefect를 설치합니다.
- Database 실습에서 실행시킨 postgresql이 잘 실행되어 있는지 확인합니다.
- `bash prefect_database_setting.sh` 명령어로 prefect의 database로 postgresql을 사용하도록 합니다.
